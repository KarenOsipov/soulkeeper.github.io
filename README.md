## 📖 О проекте

**SoulKeeper** — это платформа нового поколения для сохранения цифрового наследия. Ваши истории, тайны и послания будут жить вечно под надёжной защитой.

> *«Каждый человек — целая вселенная историй, мудрости и любви. SoulKeeper помогает сохранить эту вселенную для будущих поколений.»*

## 🛡️ Архитектура безопасности и технологии

SoulKeeper построен на принципах **Zero-Knowledge (нулевого доверия)**. Ваши данные — это только ваши данные.

*   **Сквозное шифрование (E2EE)**: Все чувствительные данные (тайны, письма, биография) шифруются на вашем устройстве по стандарту **AES-256-GCM** еще до отправки на сервер.
*   **Ключи шифрования**: Мастер-ключ генерируется из вашего пароля и **никогда не передается** на серверы SoulKeeper.
*   **Двойная защита**: Опциональный мастер-пароль для отдельных тайн. Без него доступ к ним невозможен даже при компрометации аккаунта.
*   **Firebase**: Используется только для аутентификации, хранения зашифрованных BLOB-объектов и управления доступом.
*   **Three.js**: Мощная 3D-библиотека для создания интерактивного глобуса и визуальных эффектов.
