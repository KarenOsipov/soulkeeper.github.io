## 📖 О проекте

**SoulKeeper** — это веб-платформа нового поколения для сохранения цифрового наследия и управления личными воспоминаниями. Проект сочетает в себе функции защищенного криптографического сейфа, интерактивного генеалогического древа и автоматизированной системы передачи данных доверенным лицам в случае чрезвычайных ситуаций.

> *«Каждый человек — целая вселенная историй, мудрости и любви. SoulKeeper помогает сохранить эту вселенную для будущих поколений.»*

## 🛡️ Архитектура безопасности и технологии

В основе SoulKeeper лежит парадигма **Zero-Knowledge (нулевого доверия)**. Безопасность обеспечивается на уровне исходного кода:

* **Сквозное шифрование (E2EE):** Все конфиденциальные данные (содержимое тайн, тексты писем, главы биографии) шифруются непосредственно в браузере пользователя по стандарту **AES-256-GCM**.
* **Мастер-пароли:** Ключ шифрования генерируется на стороне клиента на основе пароля пользователя и **никогда не отправляется на сервер**. Опционально для критически важных тайн можно задать дополнительный изолированный мастер-пароль.
* **Гибридное хранилище:** Приложение способно полноценно функционировать локально в браузере (LocalStorage), экспортировать зашифрованные слепки в форматы JSON/PDF, а также синхронизироваться с облачной инфраструктурой Firebase и реляционными базами данных.
